- Update the HTML page w/ the port listed for your group on the MUWB doc (this page needs the Websocket Port)

- Update the App class (the Websocket port should match the HTML doc & the HTTP port also needs to be corrected)

Doing this & running to the given bash script as given in the module should let you deploy the sample TicTacToe game to the server and map it to your group's hyperlink Note that you will need to type your group number @ the end of the bash script terminal command (i.e. bash scripts/deploy.bash group#)
